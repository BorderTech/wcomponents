Module: wc/dom/getFilteredGroup

wc/dom/getFilteredGroup

require("wc/dom/getFilteredGroup")(element, config) → {?Array.<Element>|module:wc/dom/getFilteredGroup~groupAsObject}

Encapsulates common usages of module:wc/dom/group. By default gets selected items in a group. This is changed by passing in an object containing a filter property to select items according to other states ( see module:wc/dom/getFilteredGroup.FILTERS for supported states).

Note: subgroups are honored, for example if you pass an option element the result set will be confined to sibling options within the same optgroup (if the option is in an optgroup). If not contained in an optgroup then the entire select list will be searched.

Parameters:
Name Type Argument Description
element Element | Array.<Element>

An element which belongs to (or defines) the group OR the group itself as an array.
config module:wc/dom/getFilteredGroup~config <optional>

Arguments to tweak the default behavior of this function.
Throws:

Throws a TypeError if element is false-y.

Type
TypeError
Returns:

A nullable array of elements which match the filters (or are "selected" if no custom filter supplied); or an object encapsulating both the filtered and unfiltered groups.

Type
?Array.<Element> | module:wc/dom/getFilteredGroup~groupAsObject
Examples
// the simplest usage, gets selected items
var radio = document.getElementById("radioId"),
    selectedItems = getFilteredGroup(radio);
// get disabled items in a radio group
var radio = document.getElementById("radioId"),
    disabledItems = getFilteredGroup(radio, {filter:getFilteredGroup.FILTERS.disabled});
// get items which are both selected and disabled in a radio group (silly I know)
var radio = document.getElementById("radioId"),
    disabledSelectedItems = getFilteredGroup(radio, {
        filter:getFilteredGroup.FILTERS.selected + getFilteredGroup.FILTERS.disabled
    });

<static> FILTERS :Object

An object each property of which is a bitmask representing one of the available filters. These are then used to build a filter property to pass in the configuration object module:wc/dom/getFilteredGroup~config.

Type:
  • Object
Properties:
Name Type Description
selected number

Used to filter for members of the group in any selected state.
deselected number

Used to filter for members of the group in any deselected state.
disabled number

Used to filter for members of the group in a disabled state.
enabled number

Used to filter for members of the group not in a disabled state.
hidden number

Used to filter for members of the group in a hidden state.
visible number

Used to filter for members of the group not in a hidden state.
expanded number

Used to filter for members of the group in any expanded state.
collapsed number

Used to filter for members of the group in any collapsed state.

itemWd module:wc/dom/Widget <optional>

description Describes the type of item you are looking for. This only works if the "element" parameter is a container (not an item itself and not an array). It is useful when you have a container which could contain any sort of item (like a form or a fieldset) and you want to find all selected items of a given type within that container.

Bitmask comprised of flags in module:wc/dom/getFilteredGroup.FILTERS If not provided the mask will default to "selected". Note that setting BOTH flags for a given property (e.g. hidden + visible) is the same as setting NEITHER of the flags so don't bother.
