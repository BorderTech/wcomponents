Module: wc/date/monthName

wc/date/monthName

Module which knows the names of the months for the locale in use. Month names are available as full names (for example January, janvier) and usual appreviation (for example Jan, Janv) and ascii-fied versions of these.

Requires

  • module:wc/i18n/i18n
  • module:wc/i18n/asciify

Methods

<static> get(abbreviated, asciified) → {Array.<String>}

Get the month names.

Parameters:
Name Type Argument Description
abbreviated Boolean <optional>

If true will return the standard abbreviated form of the month name. This is generally the shortest form of the month name which:

  1. Is the shortest form of the month name without being ambiguous with other month names.

  2. Is not so short that it is no longer obviously a month name
For example "O" for "October" meets #1 but not #2

Note that while this could be determined programatically we wish to allow for convention in various locales and languages.

For example in French août and avril are not abbreviated to three letters even though this would meet the criteria above.
asciified Boolean <optional>

If true will return asciified versions of the month names.
Returns:

The names of the months in order where index zero is January, index 11 is December. The returned Array is your very own special instance which you can play with to your heart's content without affecting any other users of this function.

Type
Array.<String>

<static> hasAsciiVersion() → {Boolean}

Determine if there are asciified month names.

Returns:

true if there are asciified month names (i.e. the month names differ when asciified).

Type
Boolean
