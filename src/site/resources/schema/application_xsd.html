Attributes
QName Type Use Annotation
ajaxUrl xs:anyURI required 
The URL to which the application's asynchronous HTTP requests are sent.<strong>NOTE:</strong>this attribute is under review as it is no
longer necessarily a URL different from applicationUrl.
applicationUrl xs:anyURI required 
The URL to which the application's form data is POSTed.
dataUrl xs:anyURI required 
The URL to which the AJAX requests for data lists are sent.<strong>NOTE:</strong>this attribute is under review as it is now the same as
ajaxUrl.
defaultFocusId xs:NMTOKEN optional 
The id of the component which should receive focus when the application is loaded.
icon xs:anyURI optional 
Used to replace the default favicon. Whilst this is xs:anyURI it should only be set to a URI which is able to be interpreted as a favicon by
all targeted user agents which implement favicon.
id xs:ID required 
The unique identifier for the component.
title xs:string optional 
A human readable title for the application, most commonly used for analytics.
unsavedChanges xs:boolean optional 
Set true if there are unsaved changes within the current session process; if, for example, part way through a multi-page form. This allows
client implementations to provide a warning to users if an action is likely to result in those unsaved changes being lost; for example navigation away from
the current process. Not output if "false".
