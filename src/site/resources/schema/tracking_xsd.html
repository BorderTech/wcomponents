An extention of the track type for screen level tracking. This is used in ui:application to

Attributes
QName Type Use Annotation
cat xs:string optional 
A<u>cat</u>egory indicator for the page view tracker. Some tools categorize and some don't but it is pretty common.
cd xs:string required 
<p>
  <u>C</u>ookie
  <u>D</u>omain. This attribute maps to a domain (not a URI) which is used as part of the configuration of tracking software. Each common tracking tool calls this property something different but all of them use some form of domain identifier used to set and get tracking cookies.</p>
<p>For most real uses this can be set once in the WComponent theme at compile time and forgotten. It is not likely that an application would have to switch its cookie domain at run time.</p>
clientId xs:string required 
All common enterprise level tools have some form of client identification. In most cases this identifier can be set once in the theme
at compile time and left alone but we have put this attribute in so that many applications can use a common theme and set individual client
identifiers. Obviously it could even change per page view but that would be unusual.
dcd xs:string required 
<p>
  <u>D</u>ata
  <u>C</u>ollection
  <u>D</u>omain. This attribute maps to a domain (not a URI) which is used as part of the configuration of tracking software. Each common tracking tool calls this property something different but all of them use some form of location identifier.</p>
<p>For most real uses this can be set once in the WComponent theme at compile time and forgotten. It is not likely that an application would have to switch its data collection domain at run time.</p>
name xs:string optional 
A human readable name for the trackable component. If this is not set then the page tracking will use the component's id attribute which may or
may not be desirable.

Namespace http://www.border.gov.au/Namespace/UI/V1.0
Annotations
 
<p>Tracking element is used to turn on and pass config to client side Web Analytics tools. The attributes of this element are used to set up page view tracking.</p>
