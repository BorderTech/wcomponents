QName Type Use Annotation
accessibleText xs:string optional 
Deprecated. Use toolTip instead.
buttonId xs:NMTOKEN optional 
The component id of a WButton which is used as the application submit button if the user presses 
the ENTER key with this component focussed.
decimals xs:positiveInteger optional 
The number of decimal places to show in the number field. If not set defaults to 0 to display integer values. If @step and @min are non-integer values the field may show floats (with an unqualified but user agent determined number of decimal places) if this attribute is not set or set to "0".
disabled xs:boolean optional 
The component will be in a disabled state if "true". Not output if "false".
hidden xs:boolean optional 
The component will be in a hidden state if "true". Not output if "false".
id xs:ID required 
The unique identifier for the component.
max xs:decimal optional 
The maximum allowed value in the field. Determined by the user agent if not specified and generally in the region of 0 + <largest supported long integer>.
min xs:decimal optional 
The minimum allowed value in the field. Determined by the user agent if not specified and generally in the region of 0 - <largest supported long integer>. Note that if @step is not an integer then @min must also be not an integer.
readOnly xs:boolean optional 
The component will be in a non-interactive state if "true". Not output if "false".
required xs:boolean optional 
The component will be mandatory if "true". Not output if "false".
size xs:positiveInteger optional 
The rendered width of the field in units determined by the usr agent but generally at least as large as the same number of digits. This attribute has no bearing on the number of digits which can be entered into the field.
step xs:decimal optional 
The increment to apply when the user interacts with the controls spin button nature. Defaults to 1. NOTE: @step may only be a non-integer if @min is a non-integer.
tabIndex xs:int optional 
Sets the component's position in the tab navigation order. Not implemented by default as it may 
pose significant accessibility and usability problems.
toolTip xs:string optional 
Text to display to the user as optional 'helper' information. Note, however, that there are some 
accessibility and usability implications of this attribute and it should be used with care.
track xs:boolean optional 
Indicates that the component should take part in web analytics tracking.
