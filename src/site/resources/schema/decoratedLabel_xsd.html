Element ui:decoratedLabel
Namespace https://github.com/bordertech/wcomponents/namespace/ui/v1.0
Annotations
 
WDecoratedLabel represents a piece of (usually descriptive) information which may contain other WComponents.
It is used both as a stand-alone element and as a labelling element within other components. The implementation
may have restrictions on included content which are not able to be described in this schema.
