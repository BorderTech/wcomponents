Attributes
QName Type Default Use Annotation
delay xs:positiveInteger optional
This is a delay in MILLISECONDS after which the trigger will fire (without further user intervention). Users will probably not notice any delay < 100 if there are no other pending ajax requests.
loadOnce xs:boolean false optional
Indicates the trigger may fire only once. If not true then the trigger may be fired an unlimited number of times.
triggerId xs:NMTOKEN required
The id of the element which will trigger the AJAX request.
