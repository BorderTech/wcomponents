Element ui:image
Namespace https://github.com/bordertech/wcomponents/namespace/ui/v1.0
Annotations
WImage proved a mechanism to include arbitrary images in an application. Since the applications are rendered using a web browser care should be taken to use image resources which are able to be rendered inline in the target user agent(s).

A piece of text which is able to be used as a direct, meaningful, stand-in for the image. If the image is purely decorative then this should be the empty string, otherwise it should provide the same information conveyed by the image and not be a mere description of the image.
