Attributes
QName Type Use Annotation
alt xs:string optional
Alternative text to display if the media is not able to be rendered or if the component supports in-situ images (video poster for example).
autoplay xs:boolean optional
Indicates the media autoplay setting. Not implemented by default due to accessibility concerns. Not output if "false".
class xs:string optional
Provides the ability to pass an HTML class attribute onto the root output element.
controls restriction of xs:NMTOKEN optional
Indicates what controls to display to the user. May not be implemented depending on other factors. In most cases the user agent determines the control set and only "none" has any impact. If absent, "default" is assumed and native controls will be used.
disabled xs:boolean optional
The component will be in a disabled state if "true". Not output if "false". Deprecated: HTML5 audio and video do not support the disabled state.
duration xs:positiveInteger optional
Indicates the temporal length of the media. Deprecated, not implemented by default since WComponents 12.0.001.
height xs:positiveInteger optional
The vertical size of the video player in pixels.
hidden xs:boolean optional
The component will be in a hidden state if "true". Not output if "false".
id xs:ID required
The unique identifier for the component.
loop xs:boolean optional
Indicates that the media should automatically replay at the end. May have considerable accessibility implications and should notbe used or used with extreme caution. NOTE: default implementation of this attribute may be dropped from future releases due to these accessibility concerns. Not output if "false".
mediagroup xs:NMTOKEN optional
Indicates a group name for a set of media components. Not yet implemented due to low levels of user agent support.
muted xs:boolean optional
Indicates that the media should be muted on page load. Not output if "false".
poster xs:anyURI optional
The location of an image file to display in the video player before the user loads the actual video file.
preload restriction of xs:NMTOKEN optional
Indicates the rules for pre-loading the media file(s) by the user agent. If absent, "auto" is assumed.
toolTip xs:string optional
Text to display to the user as optional 'helper' information. Note, however, that there are some accessibility and usability implications of this attribute and it should be used with care.
width xs:positiveInteger optional
The horizontal size of the video player in pixels.
