Element ui:menugroup
Namespace https://github.com/bordertech/wcomponents/namespace/ui/v1.0
Annotations
Deprecated: avoid using this component.

WMenuGroup represents a loose semantic or functional group of controls with a WMenu. For most component grouping it is preferable to use WSubMenu; otherwise separate WMenuItems using WSeparator.
