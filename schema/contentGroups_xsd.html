Element Group ui:ajaxContentGroup
Namespace https://github.com/bordertech/wcomponents/namespace/ui/v1.0
Annotations
The components which are allowed within an ajax response as a descendant of ui:ajaxtarget. This group extends contentGroup to include some elements which are not explicitly stand-along components but which are able to be individually targetted by AJAX requests.
