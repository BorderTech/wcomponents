Mozile Tests: Test Page - HTML

This is a page template for the Mozile unit test system. This particular page is HTML 4.01 Strict. The testpage.js file is used to import other test JavaScript files, which are then run in this page. Using templates, the JavaScript test files can be written once, and shared by HTML, XHTML, XML, and XUL pages.
